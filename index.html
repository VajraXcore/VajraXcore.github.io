// FAQ

Frequently asked questions.

Is VajraX really free?
Yes. All 68 tools are completely free — no paywalls, no premium tiers, no hidden costs.
Does VajraX work without internet?
60 of 68 tools work 100% offline. The remaining 8 (SSH Terminal, SNMP Browser Pro, WiFi Analyzer Pro, Network Scanner Pro, Speed Test, Wake-on-LAN Pro, Advanced Ping & Reachability Probe, DNS Lookup) require network connectivity by nature.
Do I need to create an account?
No. The app requires zero signup, zero login, and collects zero personal data, telemetry, or analytics.
Is VajraX good for CCNA/CCNP exam prep?
Yes. Includes RFC-verified subnet calculators, IPv6 tools, routing protocol simulators (OSPF, EIGRP, BGP), and comprehensive port/RFC reference databases aligned with Cisco certification objectives.
Does VajraX support multi-vendor configs?
Yes. The Universal Translator generates configs for Cisco IOS/NX-OS, Juniper JunOS, Arista EOS, and Linux Netplan.
What Android version do I need?
Android 8.0+ (API 26). The APK is approximately 4.52 MB.
Does VajraX require root?
No. All tools work without root privileges.
Can I export reports?
Yes. Every tool supports on-device export to HTML and PDF via the dual export engine. Reports are generated locally — no cloud upload.
Is VajraX safe?
VajraX is signed with RSA-4096 and APK Signature Scheme v2. It contains no tracking, no ads, and requests only necessary permissions documented at vajrax.in#privacy.
